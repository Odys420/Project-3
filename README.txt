Σκοπός του Project
Αυτό το project υλοποιεί έναν server για ηλεκτρονικό κατάστημα (e-shop) που λειτουργεί με multi-threading, επιτρέποντας σε πολλούς πελάτες να τοποθετούν παραγγελίες σε προϊόντα του καταστήματος. Ο server παρακολουθεί το απόθεμα των προϊόντων και ενημερώνει τον πελάτη για την κατάσταση της παραγγελίας του, ενημερώνοντάς τον αν το προϊόν είναι διαθέσιμο ή όχι. Κάθε πελάτης μπορεί να κάνει έως και 10 παραγγελίες και για κάθε παραγγελία, ο server ελέγχει αν υπάρχει απόθεμα για το προϊόν. Τα βασικά χαρακτηριστικά του είναι: Ο αριθμός προϊόντων στο κατάστημα όπου είναι 20, ο μέγιστος αριθμός ταυτόχρονων πελατώνόπου είναι 5, ο αριθμός παραγγελιών που μπορεί να κάνει κάθε πελάτης όπου είναι 10, η θύρα στην οποία ακούει ο server όπου είναι 8080. Η λειτουργία του είναι ο server που ακούει για εισερχόμενες συνδέσεις στην θύρα 8080, οι πελάτες μπορούν να στείλουν παραγγελίες για προϊόντα από τον κατάλογο του καταστήματος, ο server ελέγχει αν υπάρχει διαθέσιμο απόθεμα για το προϊόν που παραγγέλθηκε και επιστρέφει στον πελάτη αν η παραγγελία ήταν επιτυχής ή αν δεν υπήρχε απόθεμα.Τα αρχεία του Project είναι η main.c όπου ο κύριος κώδικας του server που διαχειρίζεται τις παραγγελίες και τις συνδέσεις πελατών και η README.txt όπου η περιγραφή του project και οδηγιών χρήσης.
